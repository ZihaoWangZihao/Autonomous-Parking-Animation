# Autonomous-Parking-Animation
# Author: Zihao Wang

# Knowledge: MATLAB, Random Walk, Split & Average Approach

# Objective: The purpose of this project is to create a visual animation of a car (illustrated as a rectangular box) randomly choosing a parking space and successfully parking itself in the spot. 

# Approach: A car has three steps to completing a park: 1) The initial forward maneuver 2) A quarter-circle turn to position itself for the space 3) The final maneuver to complete the park
